# UI  

## Base

- BGcolour: The background colour of the element.
- BGimage: A path to an image to use as the background of the element, overwrites BGcolour.

## Container  

- direction: which direction do the elements go in the container (horizontal or vertical)
- spacing: how much spacing there is between elements in the container (in pixels)
- maxElements: how many elements can there be in a line before wrapping. (-1 to disable wrapping)
